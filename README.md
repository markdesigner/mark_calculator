# caculator
這是濰昇所寫的計算機。
基本功能都有，但科學符號功能還沒實作。目前數字太長的先以四捨五入截取成跟iphone一樣的9位數
unit test 部分簡單寫了兩種case進行運算測試，以 npm run test:unit 即可運行。
